TRES HERRAMIENTAS INTEGRADAS
════════════════════════════════════════════════════════════════════════════════

🔵 MUNIN - Monitor de Métricas del Sistema
   ├─ Ubicación: backend/app/services/munin_monitor.py
   ├─ Función: Recopila 50+ métricas en tiempo real
   ├─ Métricas: CPU, Memoria, Disco, Red, Procesos, Carga
   ├─ Alertas: CPU >80%, Mem >85%, Disco >90%
   └─ Health Score: 0-100 calculado automáticamente

🟠 PINGDOM - Guardia de Disponibilidad
   ├─ Ubicación: backend/app/services/pingdom_guard.py
   ├─ Función: Verifica disponibilidad cada minuto
   ├─ Características: Health checks, uptime tracking, incident detection
   ├─ Endpoints: Configurables, extensible
   └─ Reportes: Uptime %, disponibilidad por período

🟢 SLOW QUERY LOG - Registro de Tráfico de BD
   ├─ Ubicación: backend/app/services/slow_query_log.py
   ├─ Función: Registra queries lentas e identifica cuellos de botella
   ├─ Análisis: Agrupación por patrón, impact scoring
   ├─ Umbrales: <500ms rápida, >2000ms crítica
   └─ Alertas: Queries críticas, tasa alta de lentas

CONTROL
├─ POST /api/monitoring/start       → Inicia monitoreo

CÓMO USAR
════════════════════════════════════════════════════════════════════════════════

1. Instalar dependencias
   $ cd backend
   $ pip install -r requirements.txt

2. Iniciar la aplicación
   $ python -m uvicorn app.main:app --reload

3. Acceder a los endpoints
   $ curl http://localhost:8000/api/monitoring/dashboard
   $ curl http://localhost:8000/api/monitoring/health
   $ curl http://localhost:8000/api/monitoring/alerts

4. Ver documentación
   - MONITORING_README.md (Usuario)
   - MONITORING_GUIDE.md (Técnico)
   - monitoring_examples.py (Código)
